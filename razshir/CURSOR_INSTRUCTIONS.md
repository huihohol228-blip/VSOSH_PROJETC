# Как запустить в Cursor

## Шаг 1: Запуск Flask сервера в Cursor

### Вариант 1: Через встроенный терминал Cursor

1. Откройте Cursor
2. Нажмите `` Ctrl + ` `` (обратная кавычка) или `View → Terminal` для открытия терминала
3. В терминале выполните:

```bash
cd C:\Users\gena229\Desktop\datasets\modelN
python flask_example.py
```

или

```bash
cd C:\Users\gena229\Desktop\datasets\modelN
py flask_example.py
```

4. Сервер запустится и вы увидите:
```
Запуск Flask сервера...
API доступно по адресу: http://localhost:5000
```

5. **Оставьте этот терминал открытым!** Сервер должен работать постоянно.

### Вариант 2: Создать задачу запуска (Run Configuration)

1. В Cursor откройте файл `flask_example.py` из папки `modelN`
2. Нажмите `F5` или `Run → Start Debugging`
3. Выберите "Python File" или создайте конфигурацию запуска

### Вариант 3: Использовать кнопку Run в Cursor

1. Откройте файл `C:\Users\gena229\Desktop\datasets\modelN\flask_example.py`
2. Нажмите кнопку ▶️ (Run) в правом верхнем углу редактора
3. Или нажмите `Ctrl + F5` для запуска без отладки

## Шаг 2: Установка расширения в Chrome

**Важно:** Расширение устанавливается в Google Chrome, а не в Cursor!

1. Откройте Google Chrome
2. Перейдите: `chrome://extensions/`
3. Включите "Режим разработчика" (Developer mode)
4. Нажмите "Загрузить распакованное расширение" (Load unpacked)
5. Выберите папку: `C:\Users\gena229\Desktop\datasets\razshir`

## Шаг 3: Проверка работы

1. Убедитесь, что Flask сервер запущен (в терминале Cursor)
2. Откройте любую веб-страницу в Chrome
3. Нажмите на иконку расширения
4. Нажмите "Проверить страницу"


## Отладка

Если что-то не работает:

1. **Проверьте, что сервер запущен:**
   - В терминале Cursor должно быть сообщение о запуске сервера
   - Откройте в браузере: http://localhost:5000/health
   - Должен вернуться JSON: `{"status": "ok", "model_loaded": true}`

2. **Проверьте консоль расширения:**
   - В Chrome откройте DevTools (F12)
   - Перейдите на вкладку Console
   - Нажмите на иконку расширения
   - Посмотрите на ошибки в консоли

3. **Проверьте логи Flask:**
   - В терминале Cursor будут видны все запросы к API
   - Если есть ошибки, они будут отображены там

## Полезные команды

### Остановить сервер
В терминале Cursor нажмите `Ctrl + C`

### Перезапустить сервер
1. Остановите сервер (`Ctrl + C`)
2. Запустите снова: `py flask_example.py`

### Проверить, что порт 5000 свободен
```powershell
netstat -ano | findstr :5000
```

Если порт занят, измените порт в `flask_example.py`:
```python
app.run(debug=True, host='0.0.0.0', port=5001)  # Измените на другой порт
```
