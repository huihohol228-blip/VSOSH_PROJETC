# Changelog

## [1.0.0] - 2024

### Added
- Flask веб-приложение с REST API
- Telegram бот (простой и полнофункциональный)
- Telegram Web App с красивым интерфейсом
- Chrome расширение для проверки на фишинг
- ML модели (Random Forest, Logistic Regression)
- Поддержка OCR для анализа изображений
- Парсинг .eml файлов
- Документация по развертыванию

### Features
- Проверка текста на фишинг
- Проверка изображений через OCR
- Проверка .eml файлов
- REST API для интеграции
- Множество интерфейсов (веб, Telegram, Chrome)
